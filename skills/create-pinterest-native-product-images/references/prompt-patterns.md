# Prompt and review patterns

## Product-bound prompt structure

Use this as a structure, not boilerplate to repeat unchanged:

```text
Create a new vertical 2:3 Pinterest-native editorial image.

Reference 1 is the strict product-identity reference. Preserve:
- exact object type and silhouette;
- material and construction;
- distinctive colors, markings, and parts;
- believable real-world scale.

Reference 2 is mood/composition inspiration only. Borrow:
- [lighting];
- [setting];
- [camera/composition];
- [material palette].

Scene: [one precise visual contradiction].
The image should feel like [native editorial context] before the strange
product registers. The product must remain inspectable and physically
plausible.

Do not turn the product into [likely identity failures].
No people or hands unless necessary and anatomically reliable.
No logo, packaging, text, typography, CTA, badge, price, border, collage, or
product-card frame.
```

Name the one strange thing. Generic phrases such as “make it eye-catching” or
“make it viral” do not define a visual mechanism.

## Strong hook patterns

- **Beautiful first, strange second:** an aspirational scene contains one
  ridiculous but physically credible object.
- **Serious treatment of a silly object:** photograph novelty merchandise with
  the restraint of a design magazine.
- **Functional interruption:** the product performs its real function in an
  unexpectedly elegant context.
- **Scale-correct visual punchline:** a small novelty object occupies the most
  narratively important position without becoming physically giant.
- **Specific identity:** the scene speaks to “the coworker whose desk is mildly
  unhinged,” not the generic category “office gifts.”

## Product-derived hook test

Before selecting a concept, complete both lines:

```text
The exact product feature driving this scene is: [specific shape, color,
material, marking, function, or identity].
If the product were swapped for an unrelated item, the concept would:
[break / still work unchanged].
```

Prefer a concept that breaks. A palette echo tied to the product's exact
markings or a category collision caused by its distinctive form is usually more
authored than placing any novelty object in a tasteful desk, room, or kitchen.

## Common failure modes

Reject or revise:

- a premise taken from catalog wordplay that conflicts with the visible source
  object;
- a different product type inferred from an ambiguous listing image;
- a tiny figurine rendered as a life-size animal;
- a mitt rendered as a plush toy or full creature;
- an invented pattern, material, or moving part;
- a generic beige room where the product feels pasted in;
- excessive empty luxury styling with no clear visual joke;
- floating products, product grids, badges, captions, or branded frames;
- ad copy embedded into an otherwise editorial image;
- repeated compositions with only the product swapped;
- glossy AI texture, malformed handles, teeth, eyes, or impossible object
  contact; and
- mood-reference features that override the source product.

## Revision prompt structure

```text
Revise the prior attempt while preserving [successful composition/lighting].
Correct only:
1. [specific identity, scale, material, or interaction defect]
2. [one secondary hierarchy or artifact defect]

Use the product reference as authoritative. Do not introduce a new product
shape, scene, text, branding, or additional props.
```

## Review decision

Score each dimension from 1 to 5:

| Dimension | Question |
|---|---|
| Scroll stop | Does it earn attention before copy? |
| Pinterest native | Would it plausibly appear on a saved board? |
| Aesthetic quality | Are light, materials, color, and composition intentional? |
| Concept clarity | Is one idea legible quickly? |
| Saveworthiness | Is it useful, aspirational, funny, or intriguing enough to keep? |
| Brand restraint | Does it work without overt promotion? |
| Product fidelity | Does the click destination match the pictured promise? |
| AI artifact control | Are visible generation defects acceptably controlled? |

Advance only when all hard gates pass, the mean is at least 4.0, and no
dimension is below 3. A beautiful lie fails.
